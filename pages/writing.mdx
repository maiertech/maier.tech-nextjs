# Writing guide

When I write posts for this site I follow this writing guide.

## Words to avoid

| Word   | Description                                                     |
| :----- | :-------------------------------------------------------------- |
| config | Replace with configuration.                                     |
| dev    | Replace with developer or development depending on the context. |
| repo   | Replace with repository.                                        |
| simply | Do not make things look trivial.                                |

## Special characters

### Apostrophe

Use ’ to write an apostrophe and not '. Write greengrocer’s apostrophe instead
of greengrocer's apostrophe. On a Mac use ⇧⌥\] to type ’.
